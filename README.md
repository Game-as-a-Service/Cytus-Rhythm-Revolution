# Cytus Rhythm Revolution (CRR)

## 遊戲介紹
音GAME結合TCG  
16組首性不同的歌曲組成不一樣的牌組來進行對戰

### 內容物
1. 元素盤*1
2. 16種角色，每種角色各配有四張式與七張法，共176張牌。
3. 16張腳色決鬥盤
